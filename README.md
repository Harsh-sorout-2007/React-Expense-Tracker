Expenly is a modern, highly aesthetic expense tracker application built with **React** and **Zustand**. It features a completely dynamic dark-mode user interface designed with futuristic **Glassmorphism**, neon accents, and smooth animations.

## ✨ Features

- **Premium UI:** Futuristic dark mode with transluscent frosted glass panes, responsive grid layouts, and vibrant gradient typography.
- **State Management:** Blazing fast global state management utilizing `zustand`.
- **Add Expenses:** Log new expenses by Title, Amount, Category, and Date.
- **Dynamic Filtering:** Filter expense lists in real-time by category.
- **Calculations:** Automatically calculate and present total expenditures.
- **Interactive Animations:** Entry animations for lists, hover effects for components, glowing neon borders upon input focus.

## 🚀 Tech Stack

- **Framework:** React 19
- **Build Tool:** Vite
- **State Management:** Zustand
- **Styling:** Vanilla CSS (CSS Variables, Flexbox, CSS Grid)
- **Fonts:** [Outfit](https://fonts.google.com/specimen/Outfit)

## 🛠️ Installation & Setup

1. **Clone the repository** or download the files.
2. **Navigate to the project directory**:
   ```bash
   cd ReactExpense-Tracker-main
   ```
3. **Install the dependencies**:
   ```bash
   npm install
   ```
4. **Start the development server**:
   ```bash
   npm run dev
   ```
5. **Open your browser**:
   Navigate to the URL provided in your terminal (usually `http://localhost:5173`).

## 🖌️ Design System Highlights
- **Backdrop Filters:** Uses `backdrop-filter: blur(16px)` to create stunning glass surfaces.
- **Accent Colors:** Primary actions utilize a `linear-gradient` of Cyan to Purple.
- **Responsive:** Fits perfectly on both desktop and mobile devices.
